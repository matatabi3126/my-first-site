    Claude Codeで制作 HTML / CSS / JavaScript Since 2026.07

    プログラミングは全くの未経験でしたが、Claude Codeという対話形式で指示を出せるAIコーディングツールを使い始めたことがきっかけで、このサイトを自分の手で作ってみることにしました。「こんな見た目にしたい」「こんな機能が欲しい」と会話をするように伝えながら、少しずつ形にしていきました。

    使っている技術はHTML・CSS・JavaScriptといった基本的なWeb技術で、特別なフレームワークは使っていません。バージョン管理にはGitとGitHubを利用し、公開にはGitHub Pagesという無料のホスティングサービスを使っています。ブランチを切って作業し、確認できたらmainに取り込む、という流れも少しずつ覚えていきました。

    制作はまず環境構築から始め、プロフィールと趣味を紹介する基本的なページを作って公開しました。そこから見返したりフィードバックをもらったりしながら少しずつ改善を重ね、途中でデザインを今のような雑誌風のスタイルに刷新しました。その後もスクロールアニメーションやダークモード、サイト内検索、PWA対応など、思いついた機能を一つずつ追加してきました。

    実際に手を動かしてみると、コードが読めなくても「何を実現したいか」を言葉にできれば形にできることに驚きました。一方で、公開してから気づく不具合や見た目の崩れも多く、思った通りに動かないときにどう伝えれば直してもらえるかを試行錯誤する中で、要望を具体的に言葉にする力も少しずつ身についてきた気がします。これからも、このサイト自体を育てながら少しずつ学んでいきたいと思っています。
